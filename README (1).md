# 🎛️ DJ Beto — Sitio Web Oficial

Página web profesional de **Adalberto Iglesias (DJ Beto)**, productor musical y DJ.

## 📁 Estructura de archivos

```
djbeto-site/
├── index.html          ← Página principal
├── style.css           ← Todos los estilos
├── main.js             ← JavaScript (tabs, videos, animaciones)
├── README.md           ← Este archivo
└── assets/
    └── foto-djbeto.png ← Foto de perfil
```

## 🚀 Cómo subir a GitHub Pages

1. Crea un repositorio en GitHub (ej: `djbeto-web`)
2. Sube todos estos archivos **manteniendo la misma estructura de carpetas**
3. Ve a **Settings → Pages**
4. En *Source* selecciona **Deploy from a branch → main → / (root)**
5. Guarda y en unos minutos tu sitio estará en:
   `https://TU-USUARIO.github.io/djbeto-web/`
